# 我的第一个静态网站项目

这是一个从设计到部署全流程实践的静态网站项目。

## 🚀 项目概述

本项目展示了如何：
1. 使用现代HTML5、CSS3、JavaScript技术构建静态网站
2. 实现响应式设计，适配各种设备
3. 部署到GitHub Pages（免费托管平台）
4. 实现完整的全流程开发体验

## ✨ 特性

- ✅ **现代技术栈**: HTML5, CSS3, JavaScript (ES6+)
- ✅ **响应式设计**: 完美适配手机、平板、桌面设备
- ✅ **性能优化**: 快速加载，SEO友好
- ✅ **交互体验**: 平滑动画，用户友好的界面
- ✅ **免费部署**: 使用GitHub Pages托管

## 🚀 部署指南

### 部署到GitHub Pages

1. **创建GitHub仓库**
   ```bash
   # 在GitHub上创建新仓库
   # 仓库名: my-static-website
   ```

2. **推送代码到GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/你的用户名/my-static-website.git
   git push -u origin main
   ```

3. **启用GitHub Pages**
   - 进入仓库设置
   - 找到 "Pages" 部分
   - 选择分支: `main`
   - 选择文件夹: `/ (root)`
   - 点击保存

4. **访问网站**
   - 地址: `https://你的用户名.github.io/my-static-website`

### 自定义域名（可选）

1. 购买域名（如 example.com）
2. 在域名注册商处添加CNAME记录
3. 在GitHub Pages设置中添加自定义域名

## 📱 响应式断点

- **手机**: < 768px
- **平板**: 768px - 1024px
- **桌面**: > 1024px
